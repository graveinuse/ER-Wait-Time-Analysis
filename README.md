# Hospital Emergency Room Wait Time & Patient Flow Analysis

## Project Overview
This project analyzes Emergency Room (ER) visit data to identify bottlenecks, uncover patterns in patient wait times, and build an interactive Tableau dashboard. The insights are designed to help ER operations managers optimize scheduling and resource allocation.

## Objectives
- Analyze ER patient wait times and throughput patterns
- Identify peak hours, bottlenecks, and operational inefficiencies
- Build an interactive Tableau dashboard for ER operations managers
- Provide data-driven recommendations for scheduling and resource allocation

## Project Structure
```
ER-Wait-Time-Analysis/
```

## Tools & Technologies
- **Python** — Data cleaning, transformation, and exploratory data analysis
- **Pandas / NumPy** — Data manipulation
- **Matplotlib / Seaborn** — Visualization
- **Jupyter Notebook** — Interactive analysis
- **Tableau** — Interactive dashboard

## Getting Started

### Prerequisites
- Python 3.9+
- Tableau Desktop or Tableau Public

### Installation
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd ER-Wait-Time-Analysis
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
